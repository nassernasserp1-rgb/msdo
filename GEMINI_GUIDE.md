## 🔑 **إعداد Gemini (5 دقائق):**

### **1️⃣ احصل على API مجاني:**
```
https://makersuite.google.com/app/apikey
```

### **2️⃣ الخطوات:**
1. **سجل دخول** بحساب Google
2. **اضغط "Create API Key"**
3. **انسخ الـ API key**

### **3️⃣ أنشئ ملف `gemini_config.json`:**
```json
{
    "gemini_api_key": "AIzaSyAbc123def456ghi789jkl012mno345pqr678"
}
```

## 🚀 **التشغيل:**
