# 🔑 كيفية إضافة Groq API Key

## 🎯 **خطوات بسيطة:**

### **1. احصل على API Key مجاني:**
- ادخل على: `https://console.groq.com`
- سجل مجاناً
- اذهب لـ "API Keys" 
- اضغط "Create API Key"
- انسخ الـ key (يبدأ بـ `gsk_`)

### **2. ضع API Key في الملف:**

#### **افتح ملف `groq_config.json`**
#### **ابحث عن:**
```json
"groq_api_key": "",
```

#### **ضع API key الخاص بك:**
```json
"groq_api_key": "gsk_abc123def456ghi789jkl012mno345",
```

#### **احفظ الملف**

### **3. شغل النظام:**
```bash
python laqta_ai_final.py
```
