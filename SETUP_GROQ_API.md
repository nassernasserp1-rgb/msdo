# 🔑 إعداد Groq API للنظام الاحترافي

## 🎯 **خطوات الحصول على API Key مجاني:**

### **1. التسجيل في Groq (مجاني):**
1. ادخل على: `https://console.groq.com`
2. اضغط "Sign Up" أو "Get Started"  
3. سجل بالإيميل أو GitHub
4. تأكيد الإيميل

### **2. إنشاء API Key:**
1. بعد تسجيل الدخول، اذهب لـ "API Keys"
2. اضغط "Create API Key" 
3. اختر اسم للـ key (مثل: "LAQTA-AI")
4. انسخ الـ API Key (يبدأ بـ "gsk_")

---

## 🔧 **إضافة API Key للنظام:**

### **الطريقة الأولى - ملف الإعدادات (موصى بها):**

#### **1. افتح ملف `groq_config_safe.json`**
#### **2. ابحث عن:**
```json
"groq_api_key": "",
```

#### **3. ضع API key الخاص بك:**
```json
"groq_api_key": "gsk_abc123def456ghi789jkl012mno345pqr678stu901vwx234yz",
```

#### **4. احفظ الملف**

### **الطريقة الثانية - في الكود مباشرة:**

#### **1. افتح ملف `laqta_ai_professional_safe.py`**
#### **2. ابحث عن دالة `load_groq_api_key()`**
#### **3. أضف API key في بداية الدالة:**
```python
def load_groq_api_key():
    # ضع API key هنا مباشرة
    return "gsk_abc123def456ghi789jkl012mno345pqr678stu901vwx234yz"
```

## ⚠️ **مهم للأمان:**

### **🔒 حماية API Key:**
- لا تشارك API key مع أحد
- لا تضعه في GitHub عام
- احتفظ به في ملف محلي فقط

### **🛡️ النصائح:**
- استخدم ملف `groq_config_safe.json` للإعدادات
- أو ضع API key في متغير البيئة `GROQ_API_KEY`
- النظام يعمل بدون AI (وضع احتياطي) إذا لم تضع API key
